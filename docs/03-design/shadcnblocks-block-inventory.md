## 1. Purpose

This document inventories Shadcnblocks templates, blocks, and vendor-derived UI assets used by FPDS frontend work.

Goals:
- make the current vendor footprint easy to inspect
- show where vendor assets are used in public/admin surfaces
- clarify which assets remain vendor-managed and which have FPDS wrappers
